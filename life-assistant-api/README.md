# Life Assistant API

NestJS backend for the Life Assistant platform.

For full documentation, see the [main README](../README.md) in the project root.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run start:dev

# Build for production
npm run build
```

## Development

```bash
# Run migrations
npm run migration:run

# Generate a new migration
npm run migration:generate -- src/database/migrations/MigrationName

# Lint and format
npm run lint
npm run format
```

## Environment

Copy `.env.example` to `.env` and configure your API tokens. See the main README for details.
